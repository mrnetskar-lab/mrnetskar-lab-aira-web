# GRID Hub Starter

This repo now contains the uploaded hub layout split into production-style files:

- `index.html`
- `styles.css`
- `app.js`

## Image placeholders

The UI expects portraits in:

`assets/profile_pictures/`

Expected filenames:
- `nina_profile.png`
- `hazel.png`
- `iris_profile.png`
- `vale_profile.png`

You can replace those paths in `app.js` if your real project uses a different asset structure.
